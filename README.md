App Ideas :
   -Bloc Devi is a mobile/web application that allows professionals to quickly generate block estimates based on their project dimensions.

Final Idea:
We developed this application to help professionals work faster , stay better organized , and experience less fatigue.

User Stories :
1)	Automatic calculator :
    -Input wall dimensions (length , width , height)
    -The app calculates the number of blocks needed
    -Provides the required amount of cement and sand 
    
2)	Total Cost:
  -Input the unit princes of blocks , cement , sand ,and labor 
  -The app displays the total project cost

3) Add projects:
  -Save multiple projects with their specific details 
  -Allows users to compare different estimates
 
4) PDF Estimate:
  -The app can generate a PDF estimate to send to clients 

5) Language Options(French / English)
